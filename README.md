## Recommended settings

To match the look of the theme in the screenshot use the following settings:

`Editor Settings -> Interface -> Theme`:

- Base Color: `#252525`
- Accent Color: `#569eff`
- Contrast: `0.2`
- Icon Saturation: `2`

`Editor Settings -> Interface -> Editor`:

- Main Font: [Inter](https://rsms.me/inter/)
